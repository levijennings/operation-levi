# Operation Levi

Single-file task-management web app with Supabase cloud sync, deployed on Vercel.

- `index.html` — the entire application (served at the site root)
- Backend: Supabase project `operation-levi`
- Hosting: Vercel project `operation-levi`

Editing: change `index.html`; pushing to `main` auto-deploys via Vercel.
